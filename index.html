Abstract

Robotic policies are increasingly trained on large-scale datasets collected through costly real-world interaction, yet much of this data is collected in an undirected manner, often including data that contribute little to downstream policy performance. Recent work has shown that carefully curated datasets can substantially improve policy performance, but existing approaches rely on filtering large prior datasets after they are collected, leading to wasteful data collection. In this work, we study the problem of Guided Data Collection~(GDC), which aims to actively decide what data to collect so as to maximize downstream policy performance. We introduce an iterative framework that alternates between 1)~task proposal and data collection and 2)~data evaluation. For task proposal, we leverage semantic priors from vision–language models to identify relevant environments, object configurations, and tasks for data collection. However, since semantic relevance does not necessarily imply maximal policy improvement, we perform data evaluation. By leveraging predictive data attribution we approximate how the collected trajectories influence policy performance on a target deployment setting. This signal is then used as feedback to guide subsequent data collection, closing the data collection loop. We evaluate our approach in both simulation and real-world robot settings, demonstrating that guided data collection substantially improves sample efficiency and deployment performance compared to passive data collection strategies.
